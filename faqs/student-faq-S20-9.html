What is AJAX?

AJAX allows you to make asynchronous (non-waiting) client-server requests for data. Think of it using it when you need to receive a file from a server, so you go request the file, do some other work while waiting, and when the file arrives, go and do something with that file (known as a callback function). AJAX is tricky to write so we usually do the equivalent calls in JQuery in much fewer lines of code.

What can the queued operations in JQuery be used for other than animation?

Queuing isn't an operation in JQuery, but rather it is how JQuery handles multiple animation calls.
