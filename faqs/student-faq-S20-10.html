What is WebAssembly and what is it used for?

Taken from here, WebAssembly (abbreviated Wasm) is a binary instruction format for a stack-based virtual machine. Wasm is designed as a portable target for compilation of high-level languages like C/C++/Rust, enabling deployment on the web for client and server applications. This means it's easy to do lower-level debugging and get a lot of performance gain. This is good if you've already written an app in C/C++ that you want to deploy online.

Where can I find HTML templates? Some friend of mine says that those are good for beginners?

W3 offers a lot of great templates you can try out in-browser before porting over to your site. You can even take snippets from different templates.

I'm using vs code and I have no idea where my console.log is printing to. Help?

You can view any console-logged output in the browser by right clicking on your website page in-browser and selecting the 'inspect' or 'inspect elements' option. A pannel should appear on the bottom/side, and one tab should be labeled 'console' where you can view any printed output.
