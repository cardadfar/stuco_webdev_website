How can you align a piece of text to the bottom part of a div?

This StackOverflow post provides a great response. You can set a min-height parent div, and the text inside the div you can set to absolute with a bottom of 0px. This will make it so that the space between the text and the bottom of the div is 0 so that it's bottom allisgned.

When there are multiple key frames, does the percentage in the format mean the percentage of the entire duration?

The percentage means where along the time graph the keyframe takes place. All keyframes operate on the range [0%,100%], so the percentage is a temporal value telling you at what time you'll reach the specified value. As an example, if you have a 5s animation cycle with one keyframe at 40%, then at 5*0.4 = 2s, the keyframe's value will be set.

Why do we need to know the animation stuff or how can we incorporate those into our website?

These look like two different questions. In response to the first, animation is the backbone of dynamic websites. Motion and interactivity can be used to tell clients where to click or scroll, and can be used to guide them through otherwise hard-to-traverse static sites. In response to the second, when you use an event selector such as :hover or :active in css, you can add a transition property that tells the compiler to transition between values rather than stepping between them. Other techniques involve defining keyframes or (as we will learn) assigning interpolations using jQuery.
