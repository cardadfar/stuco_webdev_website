How do you combine HTML/Javascript/JQuery together?

You can include specific script and style tags to reference the directory of your .css and .js files in your.html file. That way, when you open up your .html file, it reads from these other files. We will be covering how to do this in a future lecture.

What does changing the 'title' and the words in the p tags supposed to do on my github website? I changed those on my html file but nothing changed when I reloaded my github website.

The title tag will change the title you see at the top of your tab. The p tag is a paragraph tag that indents and provides space above and below the text. You won’t see a difference if you edit it out with only one line of text, yet if you have multiple lines, this would cause the text to merge into the paragraph above or below it. If you don’t see any changes, it could mean that you’re not pushing your files correctly. Try clicking on the file in the github repo to see if it actually updated after pushing changes.
