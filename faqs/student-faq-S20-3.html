What is the difference between HTML and CSS?

HTML is used to group different pieces of content together, while CSS allows you to style the content. This is particularly useful when you want to apply stylings to a group of elements rather than each element individually (also, as we will see, applying a property to a group is not always the same as applying the same property to each individual element in that group). Think of HTML as the wooden structure to your house that you decorate with CSS.

What are some specific examples about when you would use an ID over a class?

The biggest (and probably only) benefit to using an ID is that you can call the ID at the end of your URL, and your webpage would pre-scroll to the location of that ID. For example, if you had the website www.coolsite.com/ with an ID labeled #fun-stuff halfway down the page, then loading www.coolsite.com/#fun-stuff would start the page halfway scrolled.

I know that bootstrap is often used for more complicated elements for webpages. Are we avoiding that altogether so that we code the entire page ourselves?

Yes. It's very important to get comfortable with CSS in this class, and Bootstrap has the tendancy to make you avoid writing your own CSS by importing libraries. You get a lot more out of trying to build new properties by combining existing CSS properties than you would out of Bootstrap.

Can I define CSS class anywhere in my html file as long as it's defined before I call the class?

Yes. It is usually recommended to define it at the top since its all in one neat place for you to see.

I don't know how to adjust the font and size of the title of my webpage and text.

To adjust text, use the font-size property followed by the number of pixels (font-size: 10px). To adjust the font type, use font-family followed by the font name in qoutes (font-family: "Times New Roman"). Most basic serif and san-serif fonts are pre-loaded, although for more specific fonts, you will have to include a font file (.otf) in your directory and pre-load the font in your CSS file
